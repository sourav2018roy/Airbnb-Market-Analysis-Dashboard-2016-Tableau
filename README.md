Airbnb Market Analysis Dashboard (2016) | Tableau
Project Overview

Created an interactive Tableau dashboard to analyze Airbnb listing performance, pricing trends, revenue growth, and geographic distribution across zip codes. The project provides insights into property pricing behavior and market opportunities using data visualization techniques.

Tools Used
Tableau
Data Cleaning
Data Visualization
Trend Analysis
Geographic Analysis
Business Intelligence
Key Insights
Larger properties command significantly higher average prices.
Revenue demonstrated strong growth throughout 2016.
Most Airbnb listings are concentrated in smaller properties.
Significant pricing differences exist across zip codes.
Certain geographic regions generate higher average listing prices.
Recommendations
Increase inventory in high-performing zip codes.
Promote larger properties that generate higher revenue.
Optimize pricing strategies based on location-specific demand.
Expand supply in high-demand geographic areas.
Utilize revenue growth trends for future business planning.
